Coherence iOS

Coherence iOS is a native breathing trainer for iPhone: configurable breathing patterns, a pulsing orb, haptics, pitch-glide tones, and a daily streak system wrapped in a minimal dark SwiftUI interface.

What stands out is that this is not just a timer with nice colors. The project is trying to make a breathing session feel tactile and embodied. Visual rhythm, audio guidance, and haptic feedback all work together so the app can quietly hold the structure while the user focuses on the breath itself.

The app also feels thoughtfully composed as a native product. There is a home screen with stats and a run button, a live session state, and a settings sheet for presets, custom patterns, duration, and toggles. Underneath that are small but meaningful implementation details: an explicit breath engine, a tone engine built on AVAudioEngine, and a streak model backed by UserDefaults.
